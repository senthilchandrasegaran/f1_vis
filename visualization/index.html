Exploring F1 Statistics

An attempt at providing interactive visualizations for historical F1 data, obtained from the Ergsat developer API. This dashboard is divided into four panels, each containing interactive visualizations.

The first panel (currently containing this text) will eventually be replaced by drop-down menus that lets you select the visualization you want to explore. These visualizations will update the second panel, currently occupied by a visualization of driver rankings for the latest season.

The third panel lists all the drivers and the fourth panel lists the constructors.

Interactivity

Mouseover each driver or constructor to see how their ranking has changed over the race.
